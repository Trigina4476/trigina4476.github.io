## 🛠️ Troubleshooting

If the software does not work as expected, check these common fixes.

### The trainer does not change game values
Verify that the game process is running before you open the trainer. If the game remains unresponsive, close the trainer, right-click the file icon, and select Run as administrator.

### Antivirus software blocks the download
Some security programs flag trainers as potentially unwanted software because of how they read game memory. Create an exclusion rule in your antivirus settings for the folder containing the application. 

### Graphics settings cause crashes
If you adjust the performance sliders and experience crashes, return the sliders to their default positions. High texture settings on older hardware may lead to instability. Select the Reset button to return all values to the original game state.

### The application interface looks blurry
Windows scaling settings can cause the text to appear offset. Right-click the application, open Properties, select the Compatibility tab, and click Change high DPI settings. Check the box for Override high DPI scaling behavior.
